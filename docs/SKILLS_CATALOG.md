## 安装命令 | Installation

```bash
# 方式一：通过 ClawHub（推荐）
npm install -g clawhub
clawhub install coding
clawhub install git
clawhub install file

# 方式二：手动下载
# 访问 https://clawhub.com 搜索技能名称
# 下载后放入 ~/.openclaw/workspace/skills/

# 批量安装推荐列表
clawhub install coding git file security
```
